# 🎧 Claude para Atendimento ao Cliente

Prompts para melhorar seu atendimento, resolver problemas e fidelizar clientes.

---

## 📩 Resposta a Reclamação

```
Um cliente enviou a seguinte reclamação:
"[reclamação do cliente]"

Meu negócio: [descreva]
O que aconteceu de fato: [sua versão]

Escreva uma resposta que: reconheça o problema, peça desculpas sinceras, apresente a solução e mostre que isso não vai se repetir.
```

---

## ❓ FAQ — Perguntas Frequentes

```
Crie um FAQ completo para o meu negócio.

Produto/serviço: [descreva]
Público: [quem são seus clientes]

Liste as 10 perguntas mais comuns que clientes fazem e responda cada uma de forma clara, direta e amigável.
```

---

## 🤖 Roteiro para Atendente / Chatbot

```
Crie um roteiro de atendimento para [WhatsApp / chat do site].

Meu negócio: [descreva]
Principais demandas dos clientes: [ex: dúvidas sobre preço, entrega, cancelamento]

O roteiro deve ter: saudação, identificação da necessidade, respostas para cada situação e encerramento cordial.
```

---

## 🔁 Resposta para Pedido de Reembolso

```
Um cliente está pedindo reembolso após comprar [produto/serviço].

Motivo alegado: [o que o cliente disse]
Minha política: [aceito / não aceito reembolso / condicional]

Escreva uma resposta que seja justa, empática e proteja a relação com o cliente independente da decisão.
```

---

## ⭐ Solicitação de Avaliação / Depoimento

```
Escreva uma mensagem para pedir uma avaliação ou depoimento a um cliente satisfeito.

Canal: [WhatsApp, e-mail]
Plataforma de avaliação: [Google, Instagram, site próprio]

A mensagem deve ser curta, pessoal e deixar claro como deixar a avaliação.
```

---

## 🎁 Mensagem de Pós-venda

```
Crie uma mensagem de pós-venda para enviar após a compra.

Produto/serviço: [descreva]
Prazo de entrega/uso: [ex: produto chega em 5 dias]

A mensagem deve: agradecer, confirmar o que vem a seguir, oferecer suporte e convidar para indicar o negócio.
```
