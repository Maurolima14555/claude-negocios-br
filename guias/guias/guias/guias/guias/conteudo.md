# 📝 Claude para Criação de Conteúdo

Prompts para criar conteúdo de qualidade para blog, redes sociais e e-mail.

---

## ✍️ Artigo para Blog

```
Escreva um artigo completo para o meu blog.

Tema: [assunto]
Público: [quem vai ler]
Objetivo: [informar, converter, ranquear no Google]
Tamanho: [curto ~500 palavras / médio ~1000 / longo ~2000]
Tom: [formal, descontraído, técnico]

Inclua: título com palavra-chave, introdução que prende, subtítulos, conclusão com CTA.
```

---

## 🎥 Roteiro para Vídeo / Reels

```
Crie um roteiro para um vídeo curto (até 60 segundos) sobre [tema].

Plataforma: [Instagram Reels, YouTube Shorts, TikTok]
Objetivo: [educar, vender, engajar]
Tom: [descontraído, profissional]

Estrutura: gancho nos primeiros 3 segundos, desenvolvimento rápido, CTA no final.
```

---

## 🧵 Thread para Twitter/X ou LinkedIn

```
Crie uma thread sobre [tema] para [Twitter/X ou LinkedIn].

Minha área de atuação: [descreva]
Objetivo: [mostrar autoridade, gerar leads, engajar]
Número de posts: [5 a 10]

A thread deve começar com um gancho forte e terminar com um CTA.
```

---

## 📰 Newsletter Semanal

```
Escreva uma newsletter para meus assinantes sobre [tema da semana].

Meu nicho: [descreva]
Destaques desta semana: [liste 2-3 tópicos]
Tom: [pessoal, profissional, educativo]

Estrutura: saudação, destaque principal, seções secundárias, encerramento com próximos passos.
```

---

## 🖼️ Descrição de Produto para E-commerce

```
Escreva a descrição de produto para minha loja online.

Produto: [nome e características]
Público: [quem compra]
Diferenciais: [o que torna único]
Plataforma: [Shopee, Mercado Livre, site próprio]

A descrição deve: destacar benefícios (não só características), usar linguagem do cliente e ter um CTA.
```

---

## 🔍 Títulos Otimizados para SEO

```
Crie 10 títulos otimizados para SEO para um artigo sobre [tema].

Palavra-chave principal: [palavra-chave]
Tom: [informativo, urgente, curioso]

Os títulos devem ter entre 50-60 caracteres e incluir a palavra-chave naturalmente.
```
