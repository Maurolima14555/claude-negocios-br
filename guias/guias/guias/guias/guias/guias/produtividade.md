# ⚙️ Claude para Produtividade e Automações

Prompts para economizar tempo, organizar tarefas e criar fluxos de trabalho eficientes.

---

## 📥 Resumir Documentos Longos

```
Resuma o seguinte documento/texto de forma clara e objetiva:

[cole o texto aqui]

Formate o resumo com:
- 3 pontos principais
- Conclusão em 1 parágrafo
- Próximas ações sugeridas (se houver)
```

---

## 📧 Responder E-mails Rapidamente

```
Escreva uma resposta profissional para o seguinte e-mail:

E-mail recebido:
"[cole o e-mail aqui]"

Minha posição/resposta: [o que quero dizer em resumo]
Tom: [formal, amigável, firme]

Seja direto, educado e claro.
```

---

## 📋 Transformar Notas em Tarefas

```
Transforme as seguintes anotações em uma lista de tarefas organizadas por prioridade:

[cole suas anotações aqui]

Formato de saída:
🔴 Urgente: [tarefas]
🟡 Importante: [tarefas]
🟢 Pode esperar: [tarefas]
```

---

## 🗓️ Planejamento Semanal

```
Me ajude a planejar minha semana.

Meus objetivos desta semana: [liste]
Compromissos fixos: [reuniões, horários]
Horas disponíveis por dia: [quantidade]
Meu maior desafio de produtividade: [descreva]

Monte um cronograma diário realista com blocos de foco, pausas e tarefas prioritárias.
```

---

## 🔄 Criar Processo/SOP para Equipe

```
Crie um processo passo a passo (SOP) para a seguinte tarefa do meu negócio:

Tarefa: [descreva]
Quem executa: [cargo/função]
Ferramentas usadas: [liste]
Frequência: [diária, semanal, mensal]

O SOP deve ser simples o suficiente para qualquer pessoa nova seguir sem ajuda.
```

---

## 🤖 Prompt para Uso Recorrente (Salve e Reutilize)

```
Crie um prompt reutilizável para que eu possa usar toda vez que precisar de [tarefa recorrente].

Tarefa: [descreva]
Variáveis que mudam a cada uso: [ex: nome do cliente, produto, data]

Formate o prompt com [colchetes] nos campos variáveis para eu preencher facilmente.
```

---

## 📊 Análise Rápida de Dados

```
Analise os seguintes dados do meu negócio e me diga o que está indo bem, o que precisa de atenção e o que fazer:

[cole seus dados aqui — pode ser tabela, lista ou texto]

Contexto do negócio: [descreva brevemente]
```
