# 💼 Claude para Vendas

Prompts para fechar mais negócios, criar propostas e nutrir clientes.

---

## 🤝 Script de Abordagem Inicial

```
Crie um script de abordagem para o primeiro contato com um cliente em potencial.

Meu produto/serviço: [descreva]
Canal de contato: [WhatsApp, e-mail, telefone, LinkedIn]
Problema que resolvo: [dor do cliente]

O script deve ser natural, não parecer robótico e abrir espaço para o cliente falar.
```

---

## 📄 Proposta Comercial

```
Escreva uma proposta comercial profissional.

Cliente: [nome da empresa ou pessoa]
Problema identificado: [o que o cliente precisa resolver]
Minha solução: [o que vou entregar]
Prazo: [tempo de entrega]
Investimento: [valor]

A proposta deve incluir: apresentação, diagnóstico, solução, entregáveis, prazo, valor e próximos passos.
```

---

## 🔄 Mensagem de Follow-up

```
Um cliente demonstrou interesse mas sumiu. Escreva 3 mensagens de follow-up:

- 1ª mensagem: após 2 dias (suave, sem pressão)
- 2ª mensagem: após 5 dias (agrega valor com informação útil)
- 3ª mensagem: após 10 dias (cria urgência, última tentativa)

Produto/serviço: [descreva]
Tom da comunicação: [ex: profissional, descontraído]
```

---

## 💰 Contorno de Objeção de Preço

```
Um cliente disse: "Está caro" ou "Vou pensar".

Meu produto: [descreva]
Preço: [valor]
Principal valor entregue: [benefício]

Escreva uma resposta que reposicione o valor do produto sem dar desconto e que leve o cliente a reconsiderar.
```

---

## 📈 Upsell / Cross-sell

```
Um cliente acabou de comprar [produto A]. 

Sugira como oferecer [produto B] de forma natural e não invasiva, mostrando como o produto complementar vai potencializar o resultado que ele já vai ter.

Tom: [amigável, consultivo]
```

---

## 📋 Perguntas de Diagnóstico para Reunião de Vendas

```
Monte um roteiro de perguntas para entender melhor a necessidade do cliente em uma reunião.

Meu produto/serviço: [descreva]
Setor do cliente: [ex: varejo, saúde, educação]

As perguntas devem revelar: a dor principal, o impacto financeiro do problema, o que já tentaram fazer e o que buscam como resultado ideal.
```
